小书匠内置编辑器 codemirror 使用说明

查看pdf版本

概述

小书匠内置集成了多种编辑器，codemirror, ace,竖排编辑器，每种编辑器除了共有的提供编辑功能，还提供了比较个性化的功能

如何修改编辑器的字体

codemirror 编辑器

点击编辑区右侧的切换 编辑器按钮 ， 选择 codemirror 编辑器 就可以将当前使用的编辑器切换成 codemirror 编辑器


切换 编辑器

codemirror 设置

小书匠主按钮>设置>编辑器>codemirror


设置

行专注
